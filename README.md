## Getting Started

Get the project running on your local server.

### Prerequisites

Install the latest npm version.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/learsiOtni/macro-tracker-web-app
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
